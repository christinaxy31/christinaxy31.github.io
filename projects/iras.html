Analysis on Health Data for Medical Insurance Cost

  • Tech Stack: Python, Data Visualization, Statistical Regression, Hypothesis Testing, Causal Inference, Machine Learning, Recommender System
  • Github URL: Project Link

This study analyzed Spotify songs and ratings data to identify factors that affect songs popularity. Hypothesis Testing was conducted using Welch's T-test and Mann-Whitney's U-test, revealing that shorter and explicit songs with minor keys tended to be more popular. After applying Simple Linear Regression model, we found instrumentalness had the closest relationship with popularity. Predictive analysis using Multiple Linear Regression had a better performance with lower RMSE, while Regularized Regression Models had little improvement.

Clustering results revealed the 4 principal components identified 2 clusters, which did not correspond to the genre labels of a song and indicated that different genre labels exhibited similar characteristics.

Additionally, the study found that a single feature cannot predict mode (a binary column) well with SVM or Logistic Regression.

Finally, after preprocessing the features of songs, we built a Neural Network with appropriate hyperparameters to predict genre labels of a song. For the 52 different types of genre labels, the unweighted mean of the AUC reached 0.8869. We also built a Recommender System using popularity-based model and Latent Factor Model, obtaining a MAP of 0.3445.
